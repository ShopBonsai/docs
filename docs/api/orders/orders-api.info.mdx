# Orders API




  APIs used for interacting with orders.

  There are a few types of integration with the Bonsai Order API depending on who owns the payments.

  ## Bonsai owns the payment

  ### Native Checkout experience

  If during the checkout process, the customer is redirected to Bonsai's checkout page to complete payment and order processing.

  ### API integrated checkout experience

  If you are hosting your own checkout experience and leveraging our endpoints to collect payment and complete order processing.
  Please contact our team to provide you with all you need to get started.

  ## Account owns the payment

  If you are owning payment. Orders will need to be submitted with the payment method `external payment`.
  Once the order is submitted, our team will handle fulfillment and Bonsai will invoice you for the order amount based on our agreement.
