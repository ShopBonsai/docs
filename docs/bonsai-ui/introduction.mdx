`Bonsai UI` is a suite of Javascript tools enabling product checkout using the Bonsai platform.

Currently, we support the following integrations:

- [Express Checkout](/docs/bonsai-ui/express-checkout) <span className="label-warning">End of Life: 2023</span>
  - Single product checkout
- [Bonsai Checkout](/docs/bonsai-ui/checkout) <span className="label-success">NEW</span>
  - Multi-product checkout
  - Apple Pay and Google Pay
